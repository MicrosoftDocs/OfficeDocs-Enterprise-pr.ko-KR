> [!NOTE]

<span data-ttu-id="1af7d-140">이 명령은 Get-csonlineuser를 사용 하 여 Lync를 사용할 수 있는 다음 FederationAndPICDefault 정책을 컬렉션의 각 사용자에 게 할당 하는 Grant-csexternalaccesspolicy에 해당 정보를 모두 전송 하는 모든 사용자의 컬렉션을 반환 합니다.</span><span class="sxs-lookup"><span data-stu-id="1af7d-140">This command uses Get-CsOnlineUser to return a collection of all the users who have been enabled for Lync, then sends all that information to Grant-CsExternalAccessPolicy, which assigns the FederationAndPICDefault policy to each and every user in the collection.</span></span>
